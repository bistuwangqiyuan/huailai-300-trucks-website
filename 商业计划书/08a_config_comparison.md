# 第 8a 章 三大配置 投入产出 对比专题（v2.3 电氢完全分离 / 商业 ROI 优先 / 无氢外售）

> 引用模型：[models/02_tco_per_vehicle.csv](../models/02_tco_per_vehicle.csv) + [models/04_optimal_mix_solver.csv](../models/04_optimal_mix_solver.csv) + [models/05_fleet_capex_opex.csv](../models/05_fleet_capex_opex.csv) + [models/06_financial_npv_irr.csv](../models/06_financial_npv_irr.csv)
>
> 本章为业主 v2.1 新增 / v2.2 重写 / v2.3 再次修订专题——按 v2.3 新口径将"300 台全氢 / 200 氢 + 100 电（推荐） / 300 台全电"三种典型方案做并列对比。
>
> **v2.3 关键变化（vs v2.2）**：① **评价定位：纯商业项目 → 首要排序依据为投入产出比 / NPV / IRR / 动态回收期**（战略评分作为辅助参考）；② **制氢规模按需求反推**：A 由 36→34 MW、B 由 30→28 MW（严格匹配车队 + 10% 冗余 / 无外售）；③ **"富余氢外售"收入全部剔除**（B 方案 v2.2 的 704 万/年氢外售 → 0）；④ 副产氧气外售量按 28 MW 实际调度量调整；⑤ **不再包含"1 GW 风光电站小时级匹配测算"和"富余绿电上网"收入**（任何方案均不享受电站 交叉补贴，电站上网归电站自身）；⑥ 综合氢成本 三层叠加（基准 18 + 28 MW 制氢 CAPEX 摊销 + 加氢 + 氢险），全部重算。
>
> **口径说明**：① 三方案承担同等运输任务（240 台中途 200 km + 60 台矿区短倒，年总里程 3,357 万 km）；② 收入侧使用模型 06 基准定价（中途 0.30 元/吨·km / 矿区 0.65 元/吨·km，无任何优化叠加）；③ 全部包含 10% 氢险口径（如适用）；④ 模型 05 G2 行 电车能源成本 单位修正（4,506→451 / 1,711→171）；⑤ 制氢电费 100% 上网电价 0.40 元/kWh 采购。

## 8a.8 投入产出比 关键洞察 v2.3

### 8a.8.1 纯商业 ROI 视角（基准定价 v2.3 / 商业项目决策第一排序）

> **C 全电方案 纯财务投入产出比最高（1.29×）**：CAPEX 节省 19% + OPEX 大幅节省（无氢险无制氢电费）+ 单位运输成本 0.31 元/吨km 共同推动；简单回收期 9.1 年。**但工程可行性不通过（配电峰值 55.8 MW）→ 不能作为实施方案。**
>
> **B 推荐方案 投入产出比 0.08×（基准） → 推荐+PPA 1.35×**：基准定价 + 10% 氢险下严重亏损；叠加四项优化（西部税收 + 绿色信贷 + 服务提价 0.40/0.78 + 氢险降至 3%）+ 风光长协 PPA 0.30 → 净现值 +1.90 亿元、IRR 12.9%、动态投资回收期 7.0 年（详见第 10 章 v2.3）。**工程可行，政策红利保留，为综合最优。**
>
> **A 全氢方案 投入产出比 0.05×（基准下严重亏损，推荐+PPA 1.20×）**：制氢扩容 + 4 加氢站 + 4,614 万氢险 + 6,598 万制氢电费 负担过重，且矿区氢车单车成本 23.80 万/年远高于矿区电车 12.20 万/年 → **不推荐**。

## 8a.10 氢险率 + 风光长协 PPA 双重敏感性（v2.3 关键商业变量）
