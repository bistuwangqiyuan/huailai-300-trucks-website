> v2.3 关键变化（vs v2.2）：① **优化目标改为商业 ROI 优先**（NPV/IRR/投入产出比为第一排序依据，战略评分作为辅助参考）；② **制氢系统装机按车队需求 110% 反推 30→28 MW**，CAPEX 年化由 2,939 → 2,737 万/年（单车氢车能源全成本 -1.01 元/kg）；③ **不规划任何对外氢气销售**（v2.2 的 320 t × 22 元 = 704 万/年 氢外售收入 全部剔除 / "氢外售可对冲收益"维度删除）；④ 综合氢成本 n=200 由 49.46 → **47.94 元/kg**，单车氢车全成本 由 71.71 → **69.76 万元/年**；⑤ 明确枚举 n_氢 = 0（300 全电动）作为**纯财务 ROI 对照**；⑥ 三大配置对比详见第 8a 章。

## 8.1 单点最优解结论（v2.3 商业 ROI 优先 / 电氢完全分离 / 无氢外售）

> **n_氢能重卡 = 200 台（中途主力）+ n_电动重卡 = 100 台（含 40 中途备份 + 60 矿区短倒）= 300 台（配比 2 : 1）**

| 项 | 数值 |
|---|---|
| 氢能重卡（49吨，中途主力） | 200 台 |
| 电动重卡（49吨，40 中途+60 矿区） | 100 台 |
| 配比 | 2 : 1 |
| **综合氢成本（n=200 三层摊薄后）** | **47.94 元/kg** |
| **单车氢能重卡 综合年化全成本** | **69.76 万元/年** |
| **全队年综合成本（含 制氢/加氢/氢险 摊薄）** | **15,552 万元/年** |
| H₂ 年需求 | 2,560 吨 |
| 中途充电+换电峰值 | 8.4 MW |
| **投入产出比（推荐+PPA）** | **1.35×** |
| **商业 ROI 综合得分** | **92.0 / 100**（商业 + 政策 综合排序第一） |

> **v2.3 决策框架说明**：本章排序依据为"商业 ROI 综合得分 = 30% 投入产出比 + 20% 动态回收期 + 20% 出勤可控性 + 15% 工程可行性 + 15% 政策红利保留度"。战略评分、产业链评分等作为辅助参考，不再进入主排序。

## 8.2 综合氢成本 三层叠加摊薄模型（v2.3 口径）

### 8.2.1 三层叠加结构

| 层次 | 摊薄科目 | 数值/公式 v2.3 | 说明 |
|---|---|---|---|
| **第 1 层** | 基准氢价（业主目标变动成本口径） | 18.0 元/kg | 实际 LCOH 34.3 元/kg 的 gap 由 PPA 长协 + 制氢补贴 + 国产化 + 利用率 四路径桥接 |
| **第 2 层** | 制氢系统 CAPEX 年化 | **23,436 × 0.1168 = 2,737 万/年** | 28 MW 制氢系统全部由车队氢需求承担（v2.3 由 30→28 MW 节省 202 万/年） |
| **第 3 层** | 加氢站 综合年成本 | 757 + 594 = 1,351 万/年 | 3 座固定（n_氢 ≤ 220 不增） |
| **第 4 层** | 氢相关综合险（10%） | (3 × n_氢 + **2,998**) 万/年 | 制氢/加氢部分对 n_氢 不敏感（最大摊薄项）；28 MW 口径氢险基数下调 |

> 单 kg 综合附加 = (2,737 + 1,351 + 3 × n_氢 + 2,998) × 10000 ÷ (n_氢 × 12,800)
>                = (7,086 + 3n) × 10000 ÷ (n × 12,800)
>
> 综合氢成本 = 18.0 + 单 kg 综合附加 (元/kg)

### 8.3.1 表观最优 vs 商业 ROI 最优（v2.3）

- **纯财务 ROI 最优**：n_氢能 = 0, n_电动 = 300（投入产出比 1.65×、回收期 6.1 年）—— 但工程不可行 + 政策红利全失 + 200 km 中途电动换电运营压力大
- **表观最优（仅 综合成本 / 有可行性约束）**：n_氢能 = 180, n_电动 = 120（综合成本 15,348 万）
- **商业 ROI + 综合最优（推荐）**：n_氢能 = 200, n_电动 = 100（综合成本 15,552 万，投入产出比 1.35×）
- 200+100 较表观最优 增 204 万/年（1.3%），但 ①出勤稳定性 ②张家口示范红利 ③制氢规模 28 MW 严格匹配 ④双路线风险分散 全面占优

|---|---:|---:|---:|---:|---:|---:|

> **v2.3 结论**：200+100 商业 ROI 综合得分 92（第一），300 全电虽投入产出比 1.65× 排第一但因工程不可行被淘汰，300 全氢因矿区电车更经济 + 氢成本高而落后。

## 8.4 稳健性测试（±10% 综合氢成本 等值带 / v2.3）

| 扰动情景 | n_氢能 | n_电动（中途+矿区） | 全队 综合成本（万元） | 相对推荐方案偏差 |
|---|---:|---:|---:|---:|
| 推荐方案 | 200 | 100 | 15,552 | 基准 |
| 综合氢成本 +10%（47.94 → 52.73 元/kg） | 180 | 120 | 16,876 | +1,324（仍可行） |
| 综合氢成本 -10%（47.94 → 43.15 元/kg） | 220 | 80 | 14,441 | -1,111（仍可行） |
| 电动重卡 车辆侧 +10% | 210 | 90 | 15,782 | +230（仍可行） |
| 电动重卡 车辆侧 -10% | 180 | 120 | 15,092 | -460（仍可行） |
| **氢险率 10% → 3%（集团采购优化）** | 200 | 100 | **13,037** | **-2,515（强建议路径）** |
| **风光长协 PPA 0.30 元/kWh（vs 上网电价 0.40）** | 200 | 100 | **14,144** | **-1,408（核心商业杠杆）** |

> **结论**：推荐方案在 ±10% 综合氢成本 扰动下仍处可行域内部。**氢险率优化（10% → 3%）+ 风光长协 PPA 0.30 元/kWh 是 v2.3 两大单一最大杠杆**，合计可节省 3,923 万/年；叠加后 n=200 综合氢成本 由 47.94 → 29.5 元/kg（商业目标 18 元/kg 仍需依赖 国产化 + 示范期补贴 加持）。

## 8.6 配比的动态调整空间

虽推荐单点最优为 200+100，但项目实施过程中可保留 ±20 台动态调整空间：

| 调整方向 | 触发条件 | 调整范围 |
|---|---|---|
| 增 氢能重卡（200 → 220） | 实际 LCOH 进一步降至 < 28 元/kg（国产化加速）；氢险率降至 ≤ 3%；PPA 落地 | n_氢能 +20 |
| 增 电动重卡 矿区（100 → 120） | 工商业电价进一步下降至 < 0.35 元/kWh；中途换电站建成稳定运营 | n_电动 +20 |
| 维持 200+100 | 基准情景 | 不变 |

实施路径中，**第1年 试点期按 30 氢能重卡 + 15 电动重卡 验证两种工况，第2年-第3年 放量阶段根据试点数据微调最终配比**（详见第 13 章）。

## 8.8 推荐配比的核心论据汇总（v2.3 商业 ROI 优先 / 电氢分离 / 无外售）

| 论据 | 内容 |
|---|---|
| **工程可行性** | 300 全电配电峰值 55.8 MW 不可行 → 必须有氢车分流；200+100 配电峰值 8.4 MW 与当前电网主变余量匹配 |
| **商业 ROI** | 200+100 推荐+PPA 投入产出比 1.35×、净现值 +1.90 亿、内部收益率 12.9%、动态回收期 7.0 年 |
| **工况匹配** | 200km 中途持续作业 → 氢能重卡 占优；矿区 4km×18 趟可计划 → 电动重卡 占优 |
| **综合氢成本** | 47.94 元/kg（n=200，v2.3）经氢险+PPA 叠加 → 29.5 元/kg，商业闭环路径清晰 |
| **供能** | 制氢 28 MW → 2,816 t/年严格匹配 200 氢车 2,560 t + 10% 冗余，**无外售 / 无扩容** |
| **政策** | 200 台 氢能重卡 享 5 万/台·年 × 5 年 = 5,000 万运营补贴，3 加氢站 2,750 万建运补 |
| **风险** | 双路线风险分散；2:1 配比保留 ±20 台 灵活调整空间 |

## 8.9 本章小结

> **单点最优配比 = 200 氢能重卡 + 100 电动重卡（含 40 中途备份 + 60 矿区短倒，2:1）**
>
> v2.3 商业 ROI 优先 / 电氢完全分离 / 无氢外售 口径下：
> - 综合氢成本（n=200，三层摊薄后） = **47.94 元/kg**
> - 单车氢能重卡 年化综合全成本 = **69.76 万元/年**（vs v2.2 的 71.71 万/年；节省 1.95 万/台/年 来自 28 MW 制氢摊薄）
> - 全队年综合成本 = **15,552 万元/年**
> - 推荐+PPA 方案 **投入产出比 1.35×、净现值 +1.90 亿、内部收益率 12.9%、动态回收期 7.0 年**
>
> 这一结论基于：① 商业 ROI 综合得分 92（第一）；② 工程可行性淘汰 300 全电方案；③ 工况-车型双向匹配；④ 三层综合氢成本摊薄效应；⑤ 制氢供能 28 MW 严格匹配（无外售）；⑥ 政策红利保留度。
