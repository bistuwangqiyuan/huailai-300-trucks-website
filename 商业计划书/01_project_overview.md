# 第 1 章 项目背景与战略机遇 v2.3

> v2.3 关键变化：① **项目重新定位为纯商业项目**（追求商业投入产出比 / 净现值 / 内部收益率 / 动态投资回收期 最优），战略目标作为**辅助参考**而非主决策依据；② **1 GW 风光电站独立核算、出项目边界**，不构成本项目的资源优势叙事基础；③ 替代方案对比中 "制氢-富余可外售" 的叙事已删除；④ 制氢规模按车队需求反推至 **28 MW**。

## 1.1 业主单位概览

**怀来中基锡安新型石材科技有限责任公司**（以下简称"中基锡安"或"业主"）位于河北省张家口市怀来县土木镇，地址为怀来欣翰林文化旅游开发有限公司东 700 米。公司主营：

- 大型石材矿山开采（具备完整矿权）
- 新型人造石材深加工
- 矿区物料综合运输

公司既有资源禀赋具备稀缺组合特征：**矿权 + 1 GW 风光互补电站（独立资产，v2.3 出项目评价边界）+ 1 万亩储备建设用地**——这一组合在国内同行业中极为罕见，构成了本可行性研究的物理与制度前提。

> **v2.3 边界说明**：1 GW 风光互补电站为业主集团兄弟公司独立核算资产，按 v2.3 电氢完全分离评估原则，其发电/上网收入不计入本项目；本项目所需制氢电力与电动重卡充电电力均按电网工商业电价 0.40 / 0.55 元/kWh 公允外购，**风光电站与本项目之间零交叉补贴**。

## 1.2 项目缘起

矿区当前运输全部由柴油重卡承担，年油耗约 645 万升，年柴油成本 4,839 万元（按 7.50 元/L 计）。在以下三组外部条件叠加下，业主提出绿色化转型方案：

1. **油价波动**：2025-2026 年中东地缘冲突反复推高原油价格，柴油到岸价存在结构性上行压力
2. **国家氢能战略**：国务院《氢能产业发展中长期规划（2021-2035）》明确将"重型货运"列为氢能交通重点示范场景；2026 年三部委《关于开展氢能综合应用试点工作的通知》进一步加大示范支持
3. **张家口示范使命**：张家口为国家级燃料电池汽车示范城市群核心城市之一，按《张家口市加快建设燃料电池汽车示范应用城市群行动方案》要求，到 2025 年底推广 1,130 辆燃料电池汽车，本项目落地可一举贡献 200 辆

## 1.3 项目目标

| 维度 | 目标 v2.3 |
|---|---|
| **商业经营目标（首要 v2.3）** | **净现值 > 0；内部收益率 ≥ 8%；动态投资回收期 ≤ 8 年；投入产出比 ≥ 1.30** |
| 单位成本目标 | 全周期降低运输单位成本 ≥ 20%（低于柴油基准）|
| 战略目标（辅助 v2.3）| 矿区运输低碳化；与张家口示范区建设协同（作为辅助价值，不参与主决策）|
| 政策目标（辅助 v2.3）| 完成张家口示范城市群第二阶段 200 台 氢能重卡 推广任务 |
| **技术目标 v2.3** | **基准 LCOH ≤ 22 元/kg（PPA 情景下达成）；电网外购制氢不依赖风光补贴；矿区充氢加电出勤率 ≥ 85%** |
| 环境社会治理 目标（辅助 v2.3）| 年减排 ≥ 3.0 万 tCO₂e（v2.3 电网外购情景下）；环境社会治理 评级提升 ≥ 2 档 |

> v2.3 说明：按业主最新指示，本项目定位为**纯商业项目**，战略目标 / 政策目标 / 环境社会治理 目标均作为**辅助参考**，不作为主决策变量；主决策锚点为"商业投入产出比 / 净现值 / 内部收益率 / 动态投资回收期"。

## 1.4 战略机遇分析

### 1.4.1 政策窗口期

- **国家级**：燃料电池汽车示范期奖励（按燃料电池系统功率给付，综合奖励可达 30 万元/台）；氢能综合应用试点（2026 年新设）
- **省级**：河北省支持张家口建设国家级氢能示范区，省级配套补贴 1:1
- **市级**：张家口加氢站建设补贴 ≤ 400 万/座、运营补贴 ≤ 400 万/年/座；氢能重卡运营补贴 5 万元/台·年
- **县级**：怀来县在新能源示范区内提供土地、电网接入绿色通道

### 1.4.2 资源禀赋窗口

| 资源 | 状态 | 战略意义 v2.3 |
|---|---|---|
| **1 GW 风光电站 v2.3** | **已建成、已并网；独立核算 / 出项目边界** | **市场化 PPA 合作的潜在候选方（商业独立合同；v2.3 不作为项目绿电补贴来源）** |
| 1 万亩储备地 | 矿权范围内、土地平整成本低 | 可承载 **28 MW** 制氢厂 + 加氢站 + 储能（v2.3 按车队需求反推）|
| 矿山道路 | 自有、不受公共交通管制 | 重卡可执行非标排班、避免双班司机紧张 |
| 工业电力大用户 | 已具大用户身份 | 电网工商业电价公允购电，配电改造审批简化（v2.3 不再表述为"绿电直供制氢"）|

### 1.4.3 技术成熟度窗口

- 氢能重卡：截至 2026 年 一季度，国内 49T 氢能重卡 牵引车招标限价已降至 128 万元/台（含国补可降至 90-95 万），批量直采价（业主 v2.0 锚定）已突破 30 万元/台门槛；燃料电池系统功率主流 130-150 kW
- 电解槽：国产碱性电解槽单 MW 价格已降至 500-650 万元/MW（2026 一季度），质子交换膜电解槽 降至 800-1,000 万元/MW
- 加氢站：标准 1,000 kg/天 35 MPa 加氢站建造成本约 2,000-2,500 万元/座
- 充电桩：480 kW 液冷超充批量价已降至 60-80 万元/台

技术与价格曲线均已进入"经济性临界点"，规模化部署条件成熟。

## 1.5 替代方案对比

| 方案 | 优点 | 缺点 | 适用判定 v2.3 |
|---|---|---|---|
| 维持柴油 | 资本支出低、技术成熟 | 油价风险、碳成本上升、不符合战略 | 否决 |
| 全部 氢能重卡 (300 台) | 零碳率最高、政策红利最大 | 制氢成本过高、短倒工况浪费、商业 ROI 显著低于推荐方案 | 否决 |
| 全部 电动重卡 (300 台) | **纯财务 ROI 最优** | **工程可行性红线不通过**（200 km 中途工况下补能网络/续航/充电时间三重硬约束）| 否决（仅作财务参照） |
| **2:1 混合 (推荐) v2.3** | 工况-车型双匹配、**商业 + 政策 综合最优**、工程可行、风险分散 | 需协调两套补能体系 | **推荐** |
| 1:2 混合 | 充电友好、初期投资低 | 氢能重卡 规模不足、对示范区政策红利利用不充分 | 不推荐 |

> **v2.3 关键修订**：删除"制氢-富余可外售"与"浪费 1 GW 风光制氢潜力"两类叙事 —— 按 v2.3 电氢完全分离 + 不规划对外氢气销售原则，这两类叙事不再成立。

## 1.6 项目对张家口可再生能源示范区的贡献（v2.3 辅助价值）

> v2.3 注：本节所述战略/示范贡献作为**辅助价值**呈现，不作为本项目立项的主决策依据；业主 v2.3 明确项目首要 KPI 为商业 ROI。

- 验证"矿区低碳重卡运输"商业闭环
- 为张家口燃料电池汽车示范城市群完成 200 台氢能重卡推广任务
- 输出可复制的"矿区 + 氢电混合重卡 + 按需反推制氢"产业模型，支撑省内其他矿区复制（河北矿业大县多达 20+ 个）

## 1.7 本章小结 v2.3

- 中基锡安所处地理位置、所拥有的"矿权 + 1 GW 风光电站（独立资产）+ 土地"组合在国内具备稀缺性，叠加燃料电池示范期奖励政策窗口与油价上行的外部压力，项目具备**罕见的"天时地利"叠加条件**
- **v2.3 明确项目为纯商业项目**，首要 KPI 为商业投入产出比 / 净现值 / 内部收益率 / 动态投资回收期；战略目标 / 政策红利 / 环境社会治理 评级 均作为**辅助价值**呈现
- **v2.3 明确 1 GW 风光电站出项目评价边界**，独立核算、零交叉；制氢 / 充电电力均按电网工商业电价公允外购
- 本报告后续章节将基于详细工程测算与财务建模，验证该项目在 v2.3 严格商业口径下的 ROI 可行性
